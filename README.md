# EduClass - Sistem Manajemen Pembelajaran & Kursus Online

Aplikasi ini merupakan prototype interaktif yang dikembangkan menggunakan PHP Native, MySQL, dan Bootstrap 5 untuk memenuhi Tugas Besar mata kuliah Rekayasa Perangkat Lunak.

## Prasyarat Sistem
* XAMPP (PHP versi 7.4 atau versi 8.x)
* Browser (Chrome / Edge / Firefox)

## Cara Menjalankan Aplikasi Secara Mandiri
1. Download atau clone repositori ini ke dalam folder `C:\xampp\htdocs\`.
2. Pastikan nama foldernya adalah `educlass`.
3. Buka XAMPP Control Panel, lalu aktifkan **Apache** dan **MySQL**.
4. Buka browser, lalu pergi ke alamat `http://localhost/phpmyadmin/`.
5. Buat database baru dengan nama `educlass`.
6. Import file `educlass.sql` yang berada di dalam folder proyek ini ke dalam database tersebut.
7. Setelah berhasil, aplikasi dapat diakses melalui URL: `http://localhost/educlass/login.php`.
